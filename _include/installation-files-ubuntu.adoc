| ./debs/storagegrid-ssoauth-azure.py  a|

| ./debs/storagegrid-ssoauth-azure.js  a| 
同伴呼叫的輔助腳本 `storagegrid-ssoauth-azure.py`用於與 Azure 執行 SSO 互動的 Python 腳本。



| ./debs/extras/api-schemas  a| 
StorageGRID的 API 模式。
