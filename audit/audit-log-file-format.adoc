= 審計日誌檔案格式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
每個管理節點上都有審計日誌文件，其中包含一系列單獨的審計訊息。

每條審計訊息包含以下內容：

* 觸發審計訊息 (ATIM) 的事件的協調世界時 (UTC)，採用 ISO 8601 格式，後面跟著一個空格：
+
`_YYYY-MM-DDTHH:MM:SS.UUUUUU_`， 在哪裡 `_UUUUUU_`是微秒。

* 審計訊息本身，括在方括號內，以 `AUDT`。
