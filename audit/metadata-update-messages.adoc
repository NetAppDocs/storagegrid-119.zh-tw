== 範例：S3 元資料更新

此範例展示了成功更新現有 S3 物件的元資料的事務。



=== SUPD：S3 元資料更新

S3客戶端發出請求（SUPD）來更新指定的元數據(`x-amz-meta-\*`) 用於 S3 物件 (S3KY)。在此範例中，請求標頭包含在欄位 HTRH 中，因為它已配置為稽核協定標頭（**設定** > **監控** > **稽核和系統日誌伺服器**）。看link:../monitor/configure-audit-messages.html["配置審計訊息和日誌目標"] 。
