= 警報參考
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
此參考列出了網格管理器中出現的預設警報。您收到的警報訊息中包含建議的操作。

根據需要，您可以建立自訂警報規則以適合您的系統管理方法。

一些預設警報使用link:commonly-used-prometheus-metrics.html["Prometheus 指標"]。
