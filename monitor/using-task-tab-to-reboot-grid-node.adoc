= 使用“任務”選項卡
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]

* link:../maintain/rebooting-grid-node-from-grid-manager.html["重啟網格節點"]
* https://docs.netapp.com/us-en/storagegrid-appliances/commonhardware/placing-appliance-into-maintenance-mode.html["將設備置於維護模式"^]
